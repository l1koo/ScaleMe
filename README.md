# ScaleMe
**Плагин для изменения размера персонажа в Minecraft**.

## Особенности
*   **Анимация:** Размер игрока меняется не рывком, а плавно в течение 1 секунды.
*   **Лимиты:** Настраивайте минимальный и максимальный размер для разных донат-групп (VIP, Premium и т.д.).

## Команды и Права

| Команда | Описание | Право (Permission) |
| :--- | :--- | :--- |
| `/scale <размер>` | Изменить свой размер | `scaleme.use` (выдайте игрокам) |
| `/scale <размер> <ник>` | Изменить размер другому игроку | `scaleme.others` (по умолчанию OP) |
| `/scale reload` | Перезагрузить конфиг | `scaleme.reload` (по умолчанию OP) |
| *Нет команды* | Игнорировать любые лимиты размера | `scaleme.bypass` (по умолчанию OP) |
| *Нет команды* | Лимит из конфига (например, `vip`) | `scaleme.limit.<группа>` |

# ScaleMe
**A plugin for changing character size in Minecraft**.

<div align="center">
  <img src="https://github.com/user-attachments/assets/11648e18-fa88-4ce8-896c-06b407150d6c" alt="image" width="600" />
</div>

## Features
*   **Animation:** The player's size changes smoothly over 1 second, not instantly.
*   **Limits:** Configure minimum and maximum sizes for different donor groups (VIP, Premium, etc.).

## Commands and Permissions

| Command | Description | Permission |
| :--- | :--- | :--- |
| `/scale <size>` | Change your own size | `scaleme.use` (grant to players) |
| `/scale <size> <player>` | Change another player's size | `scaleme.others` (OP by default) |
| `/scale reload` | Reload the config | `scaleme.reload` (OP by default) |
| *No command* | Bypass any size limits | `scaleme.bypass` (OP by default) |
| *No command* | Limit from config (e.g., `vip`) | `scaleme.limit.<group>` |
